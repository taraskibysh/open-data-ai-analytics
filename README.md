# Git Fundamentals для data-проєкту

## Мета
Навчитися створювати репозиторій, робити коміти, працювати з `.gitignore`, базово використовувати гілки.

---

## Датасет
**Назва:** Вихідні документи за грудень 2025 року  
**Посилання:** [data.gov.ua](https://data.gov.ua/dataset/ffc99c1e-ac64-4b8f-ad97-858060cdb012/resource/dbfcbad0-3fc9-41b2-a5ad-5c7f25dbacda)

---

## Питання

1. Понад 30% усієї вихідної документації спрямовано до Міністерства культури та АТ «НСТУ» (Суспільне), що підтверджує роль Держкомтелерадіо як ключового органу управління та нагляду саме в цих сферах.

2. В останні 5 робочих днів року кількість вихідних документів, пов’язаних із фінансовими питаннями (Казначейство, Мінфін) та закриттям бюджетних програм, зростає вдвічі порівняно з початком місяця через необхідність завершення фінансового року.

3. Більшість вихідної пошти (понад 60%) складають стандартизовані звіти (наприклад, моніторинг заборгованості з заробітної плати), що вказує на високу завантаженість апарату бюрократичними процедурами порівняно з індивідуальною проєктною роботою.
