# Outgoing Documents for December 2025

## Dataset Description
This dataset contains information about **outgoing documents for December 2025** within the system of record management of the **State Committee for Television and Radio Broadcasting of Ukraine**.  
It is part of a broader dataset that describes the **accounting system and types of information stored** by the committee.
